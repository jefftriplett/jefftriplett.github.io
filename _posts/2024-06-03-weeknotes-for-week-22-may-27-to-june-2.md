Several projects gained [EchoFeed](https://micro.webology.dev/2024/05/31/on-echofeed.html) support, making publishing an RSS/ATOM feed to Mastodon easier.

[Upgrade Django](https://upgradedjango.com): We had four updates this week, which is rare.

I wrote a GitHub Issues sync tool that takes the open issues in a repo and saves them locally to markdown + front matter. The tool can also sync changes back.

I wrote a sitemap research tool.
